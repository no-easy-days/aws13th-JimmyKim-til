## 📝 배운 내용 (Today I Learned)

## Python 실습 과제 막혔던 부분들 정리

 1. 스코프(Scope)
   - 변수의 범위를 뜻하는 용어. 지역변수 or 전역변수
 2. LEGB 규칙
   - Python의 변수 찾는 규칙. Local(지역변수) →Enclosing(외부함수의 지역변수)→Global(전역변수)→Built-in(파이썬 내장)
 3. 람다 함수(lambda)
   - 한번 쓰고 버릴 간단한 함수를 별도 선언없이 만들 수 있다. 일종의 "포스트잇 메모", 한줄 표현식만 가능!
   - 다른 함수에 "인자값을 전달할 때 유용하게 사용"할 수 있다. 딱 한번만 쓸 수 있다.
   - 사용 방법 → "lambda 매개변수 : 반환할표현식" (매개변수는 있거나 없거나.)
   - 사용 예시)
    ```
   	square = lambda x : x ** 2
    ```
4. 재귀함수(Recursive)
   - 함수 안에 자기 자신을 다시 호출하는 것, Python은 기본적으로 약 1000회까지만 재귀 호출을 허용
   - 계층적/트리 구조, 수학적 정의와 비슷할 때 사용하면 좋다.
   - "기저조건"이 반드시 필요하다. 재귀를 멈추는 상황.
   - 사용 예시)
    ```
      def factorial(n):
          if n <-= 1:
                return 1
          return n * factorial(n-1)
    ```
5. 리스트 컴프리헨션(List Comprehension)
   -  "직관적"이고 "간결하게" 리스트를 생성한다. 대괄호[]로 감싸고 내부에 for, if else문을 활용한다.
   -  대괄호 내에서 수식 또는 함수, 조건문 적용이 자유롭고 매우 편리하다.
   -  사용 방법 : [ 표현식 for 변수 in iterable ]
6. map() / filter()
   - 여러개의 데이터를 담는 자료형에 대하여 모든 요소에 함수를 적용하도록 설정 / 조건에 맞는 요소들만 추출
   - "이터레이터(iterator)"를 반환하기 때문에, 리스트 또는 튜플과 같은 자료형으로 저장해둬야한다.

### 이터레이터(iterator) & 이터러블(iterable)
   1) 이터러블
      - 반복 가능한 "대상"을 가리킴, 이터레이터를 만들어 낼 수 있는 것들
      - 대표 예시 : 리스트, 튜플, 스트링, 딕셔너리, 셋, 레인지 등등
   2) 이터레이터
      - 반복을 실제로 수행하는 객체를 가리킴, "소비자"
