### AWS 기초
- 공인 교육 포함해서 네트워크 및 인프라 기초 개념을 집약해서 구성해보자.
  - Public Cloud : AWS, GCP, Azure
  - Private Cloud : ***OpenStack***을 활용해서 구축 가능.
- IAAS(Infrastructure As A Service) → PAAS(Platform As A Service) → SAAS(Software As A Service), 이 중에 IAAS 부터.
  - IAAS : Bare-Metal 서버를 제공한다. 직접 OS 선택하고 설치
  - PAAS : OS 보다 더 윗단으로 생각하자. Docker 처럼 컨테이너 개념으로 받아들여도 됨
  - SAAS : 클라우드 환경에서 돌아가는 프로그램. Google Docs 같은 거.
- 결국 "과금"이 중요... 또나오는 3단어 → ***"Pay-As-You-Go, Internet, On-Demand"***
- 당일 사용 요금은 익일 새벽 4~5시 경 확인 가능하다. 매일 아침마다 확인하는 습관을 들이자.
