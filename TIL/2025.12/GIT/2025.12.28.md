## 📝 배운 내용 (Today I Learned)

### 변수와 자료형 (int, float, str, bool, list, dict, tuple, set)
#### ※ 자료형이란?
  - Python을 포함한 프로그래밍 언어에서 변수 또는 값이 가질 수 있는 데이터의 종류를 뜻한다.
#### (1) int
  - 숫자형 자료형, **정수** 형태
#### (2) float
  - 숫자형 자료형, **실수** 형태. 소숫점 및 컴퓨터식 지수 표현 방식이 포함된다.
#### (3) str
  - 문자열 자료형
#### (4) bool
  - 불 자료형, 참(True) 또는 거짓(False)을 나타낸다.
#### (5) list
  - 여러 데이터들을 묶어서 관리하기 위한 자료형이며, **대괄호[]** 로 표현한다.
#### (6) tuple
  - 여러 데이터들을 묶어서 관리하기 위한 자료형이며, **소괄호()** 로 표현한다. 리스트와 비슷하지만 가장 큰 차이점은 요소값 변경이 불가능하다.
#### (7) dict
  - 키&밸류(Key & Value) 쌍 형태이며, 일종의 대응 관계를 활용할 수 있는 자료형이다.
#### (8) set
  - 수학의 **"집합"** 과 동일한 개념이다. **중복 허용 X, 순서가 없다**는 점이 특징이다. 

### 조건문과 반복문 (if, for, while)
#### if 문
  - 조건 제어문
    ```
    // if 문의 기본 구조
    if 조건문:
    수행할_문장1
    수행할_문장2
    ...
    else:
    수행할_문장A
    수행할_문장B
    ...
    ```

#### while 문
  - 반복문, 주어진 조건이 참인 동안 계속해서 수행하는 형태 
    ```
    while 조건문:
    수행할_문장1
    수행할_문장2
    수행할_문장3
    ...
    ```
    
#### for 문
  - while과 비슷한 반복문이지만, 조건 및 구조가 한눈에 들어온다는 것이 특징.
    ```
    for 변수 in 리스트(또는 튜플, 문자열):
    수행할_문장1
    수행할_문장2
    ...
    ```
    - list 또는 tuple 또는 문자열의 첫 번째 요소부터 마지막 요소까지 차례로 변수에 대입해서 수행.
   
### 함수 정의와 호출
#### 함수란?
  - 특정한 기능을 실행하거나, 그 특정한 기능을 재사용 가능하도록 구성하는 코드 블록
    1) 함수의 구조
       ```
       def 함수_이름(매개변수):
       수행할_문장1
       수행할_문장2
       ...
       ```
    2) 함수 호출 시 필요한 유의 개념
       - 매개 변수(parameter, 파라미터) : 함수에 입력으로 전달된 값을 받는 **변수**
       - 인수 (arguments, args 등등) : 함수를 호출할 떄 전달되는 **입력 값**
    
### 기본 입출력 (input, print)
#### input
  - 사용자가 키보드로 입력한 모든 것을 **문자열(str) 형태**로 저장.
    ```
    a = input()
    ```

#### print
  - 크게 문자열 / 변수 값 / 지정 형식 등으로 입력받은 것을 출력한다.
