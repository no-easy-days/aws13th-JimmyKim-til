## Python 실습 과제 막혔던 부분들 정리
  - 자동완성 기능 껐다. 공부가 안되는 느낌이라... 
  - Pycharm에서도 GitHub repo에 "바로" commit을 할 수 있는 기능이 있는 것 같다.
    1) 실제로 연동하고, 인증 작업까지 진행했는데, push가 안된다. 뭔가 인증에 문제가 발생했다.
       - Token을 생성하고 인증했더니 된다. 권한 부여가 문제였던거 같다. 자세한건 잘모르겠지만...

  1. split 명령어
     - 변수를 하나 할당해서 split을 처리하자.
  2. range 명령어
     - 특정 구간의 숫자 범위 만드는 함수.
  3. 리스트 끼리 합치길 원할 때
     - 그냥 두가지 리스트에 연산자 + 하면 된다.
  4. set 자료형
     - 교집합(&), 합집합(|), 차집합(-)이 가능하다.
  5. dictionary 자료형
     - keys() / values() : 소괄호 안에 아무것도 안넣으면 각각의 리스트로 뽑아낸다.
     - items() : 키 밸류 쌍으로 뽑아낸다.
  6. enumerate(a, start=1) 함수
     - 번호를 매기다. 리스트 a항목의 번호를 1부터 매긴다.
  7. zip() 함수
     - 리스트 밸류 값을 하나씩 짝 지어준다.
