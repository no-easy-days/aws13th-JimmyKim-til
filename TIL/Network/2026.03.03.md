### SSH - secure shell
- 원격 접속 프로토콜. 암/복호화 처리된 메시지를 송수신한다. TCP 22번 포트 사용.
  ※ Telnet : 평문 형태의 메시지 송수신 프로토콜. TCP 23번 포트 사용
- 인증 방식
  1. ID & PW
  2. 키 페어 → 공개 키 & 개인 키로 구성되어 있으며, 비대칭 형태!
     - Private Key(개인 키) : 클라이언트 단 보유, ***"열쇠"***
     - Public Key(공개 키) : 서버 단 보유, ***"자물쇠"***
  3. 암호화
     - 메시지를 어떻게 암/복호화할 지를 정의한다. AES 대칭키 알고리즘이 일반적이고, ***세션을 맺을 때 활용***.
