### VyOS
- Vyatta의 무료 버전, ***Zone 기반 방화벽 기능***이 있는 오픈소스형 Router 운영체제.
※ VyOS 설치 및 설정 초기화
   1. Default 계정 & PW 로그인 및 OS 설치
      ```
      기본 설정된 root ID : vyos / root 비밀번호 : vyos 로 로그인
      install image 입력 (메뉴 뜨는 대로 옵션 입력)
      ```
   2. 기본 명령어
      ```
      ① conf : 관리자 설정 모드 진입
      ② set : 설정
      ③ show : 조회
      ④ delete : 삭제
      ⑤ edit : 수정
      ⑥ ★commit : 반영★ ~ 설정 마치고 항상 커밋 해줘야한다! 
      ```
   3. 메인 이더넷에 IP 주소 부여
      ```
      set interface ethernet eth0 address <IP주소/Prefix>
      set system gateway-address <게이트웨이주소>
        ＃ 여기서 게이트웨이는 Default Route로 활용된다.
      commit
      run show interface 
      ```
#### 항상 Tab을 치면서 명령어를 완성하자. 결국 리눅스 기반이다.
#### 설정값을 잘못 입력했을 경우 set을 한번 더 입력하면 된다.


### VyOS에서 NAT 설정하기.
<img width="688" height="522" alt="image" src="https://github.com/user-attachments/assets/7c0d957a-ba05-45a3-8787-5425ae5ed4d5" />


※ PAT 설정하기 (내부 → 외부, OutBound = Egress)
   1. PAT를 적용한 네트워크 주소 선언
      ```
      set nat source rule <룰 번호> source address <네트워크주소/Prefix>
      ```
   2. 외부로 나갈 인터페이스 이더넷 정의
      ```
      set nat source rule <룰 번호> outbound-interface <이더넷ID>
      ```
   3. 외부로 나갈 인터페이스 이더넷에 오버로드
      ```
      set nat source rule <룰 번호> translation address masquerade
                                                           ↑
                                         cisco Router에서 overload랑 같은 의미
      ```
   4. 적용 결과 조회 or 잘못 입력한 설정값 삭제
      ```
      - commit
      - show nat source
      - delete nat source rule <룰 번호>
      ```


※ DNAT 설정하기 (외부 → 내부, Inbound = Ingress)
   1. 외부에서 들어올 인터페이스 이더넷 정의
      ```
      set nat destination rule <룰 번호> inbound-interface <이더넷ID>
      ```
   2. 프로토콜 정의
      ```
      set nat destination rule <룰 번호> protocol <프로토콜 명칭, 웬만해서는 tcp 겠지.>
      ```
   3. 내부 서버의 주소 정의
      ```
      set nat destination rule <룰 번호> translation address <내부 서버 IP 주소>
      ``` 
   5. 내부 서버의 포트 정의
      ```
      set nat destination rule <룰 번호> translation port <포트 번호>
      ```
   6. 반영 및 적용 결과 조회
      ```
      - commit
      - show nat destination
      ```
