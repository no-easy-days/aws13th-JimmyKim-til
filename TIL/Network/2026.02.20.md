### Linux에서 DNS 서버 구축하기
1. hostname 변경하기
   - hostnamectl set-hostname <원하는이름>
  
     
2. DNS 패키지 설치
   - yum install -y bind bind-utils
     ※ bind : DNS Server를 구성할 수 있는 모듈이 담긴 패키지 / bind-utils : DNS 클라이언트 역할을 구성할 수 있는 모듈이 담긴 패키지

     
3. 수신 포트 및 허용 대상 설정
   - vi /etc/named.conf
     <img width="1140" height="478" alt="image" src="https://github.com/user-attachments/assets/03bb2165-01f7-479d-94f3-5b5a3582c119" />
     ※ 원래는 허용하는 IP 와 Port를 지정해야 하지만, 우리는 연습용이니까 일단 any로.

     
4. 영역(Zone) 생성
   - vi /etc/named.rfc1912.zones
     ※ 이건 규약처럼 정해진 것이기 떄문에, 추가 및 수정해서 사용해나간다.
     <img width="902" height="626" alt="image" src="https://github.com/user-attachments/assets/ac741c1c-d664-4ca4-a67a-c1e8e4e7c4cd" />


5. 영역(Zone)에서 할당한 대상 파일 생성
   - vi /var/named/<대상이름>
  
     
   ```
    $TTL 3H
    @               SOA     @       root (
                                                      0       ; serial
                                                      1D      ; refresh
                                                      1H      ; retry
                                                      1W      ; expire
                                                      3H )    ; minimum

                    IN      NS      @

   ```
