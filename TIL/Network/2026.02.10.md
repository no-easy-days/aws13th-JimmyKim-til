## 📝 배운 내용 (Today I Learned)


### CIDR(Classless Inter-Domain Routing)

- 낭비가 심한 Classful Network의 대항마. ***서브네팅*** 기법을 통해 Classless, 효율화한다.
- 서브네팅 : 주어진 네트워크 대역을 여러 개의 네트워크로 쪼개는 행위.
- 한 옥텟에서 서브네팅 결과 검산 방법 : 256 - Prefix = 네트워크 규모, 서브넷 마스크에서 1은 "Fix"를 의미함을 잊지 말자.

### Cisco Packet Tracer 실습

- ***스위치 : 같은 스위치에 물려있으면 그것은 같은 네트워크에 있다는 것을 의미한다!***
- ***라우터 : 라우터를 기준으로 네트워크가 분리된다!***

1. 주요 명령어
   - en : 관리자 모드 진입
   - conf t : 전역 설정 모드 진입
   - interface f0/0 : fastEthernet 0번슬롯/0번포트로 할당된 인터페이스에 대한 설정 모드 진입
   - ip address <ip주소> <서브넷마스크> : 해당 인터페이스에 IP 주소 할당
   - ***no shutdown : 설정 완료한 인터페이스 논리적 활성화. 꼭 해줘야 됨! 그리고 장비 재부팅 시 꼭 다시 해주기!!!***
   - do show run : 모든 설정값 확인. 스페이스 눌러가며 확인한다.
   - do show ip interface brief : 대상 장비의 인터페이스에 할당된 ip 정보를 간략하게 조회

#### Default Gateway : 목적지가 어디든 간에, 내가 속한 네트워크를 빠져나가기 위한 관문 → 그렇기 때문에 네트워크~라우터 인터페이스를 생각하자.


#### 네트워크가 적절히 구성되었다는 의미
1. 내가 속한 네트워크망에 포함 되도록 적절한 대역(IP)를 사용해야 한다.
2. 적절한 장비에 연동되어야 한다. → 물리적으로 연결되어 있어야 한다.
